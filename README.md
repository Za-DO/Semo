
# SETLIST

[App Store 바로가기](https://apps.apple.com/us/app/setlist/id6443660710)

여러분이 부르고 싶은 노래, 기억하고 싶은 노래 셋리스트에 모두 담아보세요!

SETLIST는 노래방에서 부르고 싶은 노래, 연습하고 싶은 노래를 보다 쉽고 자세하게 기록할 수 있는 앱입니다.
